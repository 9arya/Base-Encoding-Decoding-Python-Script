# BEDPS

Base-Encoding-Decoding-Python-Script

adalah kumpulan script untuk menggunakan encoding seperti base64 menggunakan library base64.

untuk memakainya kamu harus menginstall beberapa paket


1.Python3.12 atau lebih tinggi

2.git


untuk mengunduhnya gunakan perintah
```bash
git clone https://github.com/9arya/Base-Encoding-Decoding-Python-Script.git
```
didalamnya ada beberapa file dan fungsinya berbeda beda.


fungsi.py adalah sebuah module untuk main.py yang dimana ini berbasis input.
```bash
python main.py
```


ada juga cli.py yang berfungsi sama seperti main.py namun berbasis cli.
gunakan dengan:
```bash
python cli.py en b64 hallo semuanya
```


ada juga enf.py dan def.py yang digunakan untuk meng-encode atau meng decode file.
```bash
python enf.py input.file output.flie
```


lalu layer_encode.py berfungsi untuk meng-encode secara berlapis.
```bash
python layer_encode.py encode hallo semuanya
```

NOTE:jangan digunakkan untuk hal yang negatif seperti menyontek atau mencela orang lain
